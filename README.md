# IP9
What is my IP address

#### Setup
Install flask:
```bash
pip install Flask
```

Clone this repository:
```bash
git clone git@github.com:MParvin/IP9.git
```

Run it:
```bash
python flask_app.py
```

Now, you can use it with browser or cli:
```bash
firefox localhost:8080
```
or
```bash
curl localhost:8080
```

I'll deploy it on IP9.ir, so you can use it without setup:
```bash
curl ip9.ir
```
